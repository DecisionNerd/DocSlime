# Design

This folder holds product design and style guidance behind the user experiences.

## What lives here

docgen is a CLI, so it does not currently need a visual design system. The folder exists as
the standard place for UI, brand, accessibility, content, interaction, and component guidance
in projects that do have a user-facing product surface.

- **[style-guide.md](style-guide.md)** — the default design, content, interaction, accessibility, and component guidance.
- **Additional design docs** — focused deep dives when the style guide grows too large.
